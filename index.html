① 今日重点 Brief Cards

High PriorityEmbodied AI

宇树等 humanoid/legged robotics 公司进入资本化与规模化验证阶段

行业从 demo 转向 mass production、供应链、可靠性和部署证据。对研究者来说，real-robot validation 的价值继续上升。

展开：对你意味着什么

Research implication：你更适合走 real-robot embodied autonomy，而不是纯模型追热点。

实验条件匹配：六足机器人 + depth sensing + cluttered navigation 可以形成稳定系统论文路线。

Medium-HighBig-shot Opinion

Agent 系统工程从 Prompt Engineering 走向 Harness / AIWare Engineering

可用 Agent 依赖 router、memory、context compression、tool orchestration 和 cost control。

展开：研究方向映射

机器人系统同样需要分层架构：high-level intent + low-level safety/controller。未来可以做 command-space router 或 risk-aware embodied agent。

High PriorityPaper

VLM / VLA 的鲁棒性问题仍是具身智能闭环控制风险

VLM 在视觉推理、CoT faithful reasoning、误导上下文下仍可能不稳定。闭环控制不宜完全交给 language reasoning。

展开：可发表问题

可以研究：VLM high-level intent + safety-aware local arbitration。该结构比 end-to-end VLA 更符合你现有实验平台。

MediumResearch Trend

视频 tokenization / long-horizon visual memory 会影响 robot world model

VideoFlexTok 等工作指向更高效的视频表示。对机器人来说，关键是长时序记忆、动作理解和环境变化建模。

展开：与你平台的关系

短期不建议硬做 world model；更现实的是 local depth map + short obstacle memory + failure analysis。

High PriorityArbitration

Multi-agent system 暴露“专家意见被平均化”的问题

多个 agent 不一定强于单个专家，关键在 expertise routing 和 conflict resolution。

展开：研究启发

这直接支持一个大方向：conflict arbitration。机器人里 Follow/Avoid、goal/safety、semantic/geometric 都需要仲裁机制。

High PriorityCapability

头部 robotics 公司能力要求：robot learning + control + deployment

核心能力不是只会训练模型，而是能把 policy、perception、planning、control、ROS/C++/Python 和真机调试串起来。

展开：你的补齐路线

优先级：ROS2/Nav2 → C++ 工程 → MPC/CBF/TEB/DWA → RL sim-to-real → data pipeline & telemetry。

③ Capability Gap Matrix

短期必补Baseline credibility：DWA / TEB / MPC / CBF-QP、统计检验、failure taxonomy
中期必补ROS2、C++、experiment automation、log analysis、real-robot debugging
长期加分RL sim-to-real、VLA/VLM high-level planning、dataset/benchmark construction
产业映射Robot learning engineer、robotics software engineer、system integration / validation engineer

④ What should I do next

  1. 把现有平台整理成 repeatable evaluation benchmark：场景、指标、失败类型、日志模板。
  2. 补一个可信的 local planner baseline：DWA / TEB / MPC / CBF-QP 至少选一条做扎实。
  3. 每周读 2 篇 embodied AI / robot learning 论文，只提炼：可复现实验、可迁移到六足平台的点。
  4. 工程侧开始补 ROS2 + C++ + experiment automation，把“会跑实验”升级为“会搭系统”。
